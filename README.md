# EL ULTIMO PARCIAL

Este proyecto contiene el videojuego “El Último Parcial” desarrollado en Unity.
Sigue estas instrucciones para abrirlo, ejecutarlo y compilarlo correctamente.

## Requisitos

Antes de abrir el proyecto necesitas:

Unity Hub

Unity Editor 2021.3 LTS o superior

(Opcional) Visual Studio / VS Code para editar scripts

## Ejecucion

1. Clona o descarga el repositorio.

2. Abre Unity Hub.

3. Haz clic en Open.

4. Selecciona la carpeta raíz del proyecto: `/ElUltimoParcial/`

5. Espera a que Unity importe los assets.

## Ejecutar el Juego en el Editor

1. Abre la escena del menú:

'Assets/Scenes/MainMenu.unity'

2. Presiona el botón Play (arriba en Unity).

3. Usa el menú:

   - Play → inicia el juego

   - Exit → cierra la aplicación
